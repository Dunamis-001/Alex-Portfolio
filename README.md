# Alex 'Dunamis' Nyamai - Portfolio

Digital Strategy & Brand Growth Portfolio

## About

This is the professional portfolio website for Alex 'Dunamis' Nyamai, showcasing expertise in digital strategy, brand growth, media leadership, and software development.

## Features

- Clean, modern design
- Responsive layout for all devices
- Interactive project cards
- Direct links to all projects and social media
- Professional presentation of skills and experience

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The portfolio will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 19
- TypeScript
- Vite
- CSS3

## Structure

- `src/App.tsx` - Main application component
- `src/data/portfolio.ts` - Portfolio data (projects, skills)
- `src/App.css` - Styling
- `index.html` - HTML template

## Customization

To update content, edit `src/data/portfolio.ts` with your project information.
